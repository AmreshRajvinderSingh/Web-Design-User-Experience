This is a readme file for my portfolio which has all the tag which i have used.

<!DOCTYPE html>:This declaration defines the document as an HTML5 document.
<html>: This is the root element of an HTML document.
<head>: Contains meta information, links to stylesheets, and other resources for the web page.
<meta>: Provides metadata about the HTML document. Metadata isn't displayed on the page but is machine-readable.
<title>: Specifies the title of the webpage, which is displayed in the browser's title bar or tab.
<link>: Defines the relationship between the current document and an external resource. Commonly used to link to stylesheets.
<body>: Contains the content of the webpage.
<header>: Represents a container for introductory content or navigation links. Typically contains a group of introductory or navigational aids.
<nav>: Represents a section with navigation links.
<ul>: Represents an unordered list of items.
<li>: Represents a list item, typically used inside ordered (ol) or unordered (ul) lists.
<a>: Defines a hyperlink, which can be used to link to another document, an external site, or another section within the same document.
<section>: Represents a standalone section in a document.
<div>: A generic container for flow content. It does not inherently represent anything but can be used for styling purposes.
<img>: Embeds an image in the document.
<p>: Represents a paragraph of text.
<h2>: Represents a level 2 heading, with <h1> being the highest and <h6> being the lowest.
<table>: Represents a table.
<tr>: Represents a row in a table.
<th>: Represents a header cell in a table.
<td>: Represents a data cell in a table.
<main>: Represents the main content of the document, excluding headers, footers, and navigation.
<figcaption>: Represents a caption or legend for a <figure> (not used in this code) or for a part of the <figure> like an image.
<footer>: Represents the footer of a document or section, typically containing information about the author, copyright information, or related documents.
<summary>: Represents a summary, caption, or legend for a <details> element (not used in this code).
<form>: Represents an HTML form that collects user input.
<div>: A generic container for flow content, which does not inherently represent anything. It can be used for styling and layout purposes.
<label>:Represents a caption for an <input> element.
<input>: Represents an input field. Its type can be specified with the "type" attribute (e.g., text, email).
<textarea>: Represents a multiline text input control where users can enter text.
<button>: Represents a clickable button.
<video>: Embeds video files in the document and allows for playing them.
<source>: Specifies multiple media resources for elements such as <audio>, <video>, and <picture>. Allows you to specify different media files for different devices or screen sizes/resolutions.
<audio>: Embeds audio content in a document, which can be played by users with controls for play, pause, etc.
